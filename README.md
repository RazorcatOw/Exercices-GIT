# MA première modif !
exercices git pour apprendre comment ça fonctionne
On apprend plein de choses sur GIT !

J'ai utilisé mkdir pour créer un dossier sur mon bureau
Ensuite j'ai fait cd GitHub pour aller dans ce dossier
J'ai regardé ce qu'il y avait dedans avec ls
J'ai cloné avec git clone mon projet github grâce au lien fourni sur le site
et là j'ai fait "code ." pour modifier via VS code mon README.md

2e partie : j'ai fait un clear pour mettre mon terminal au propre
j'ai fait un git status pour voir que mon fichier avait été modifié sur mon pc, mais pas sur GIThub !
j'ai ensuite utilisé git add README.md
puis git commit -m "Changement du README.md pour annoncer ce que j'avais fait
et enfin, j'ai terminé par git push
